Order Plugin – Advanced SMP Economy System

A highly optimized and lightweight Minecraft Order Plugin built for modern SMP servers. Designed with Folia compatibility, seamless economy integration, and performance in mind.

✅ EssentialsX Economy Support – Works perfectly with Essentials economy.
✅ High Performance Optimization – Minimal lag impact and optimized task handling.
✅ Modern Architecture – Clean and efficient system for stability.
✅ Lightweight & Fast – Designed to run smoothly even on larger servers.
